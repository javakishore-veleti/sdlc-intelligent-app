# sdlc-admin

Angular portal for **administrators**.

The administrative UI for managing the SDLC master data:

- Register and manage **business applications** (microservices / projects).
- Define **sprints** and their **agendas** under each application.

This is the write/setup surface of the platform. The data defined here is what the
`sdlc-nexus` portal lets customers browse and query.

Backed by `Middleware/MasterDataAPI`. See [`../../PRD.md`](../../PRD.md).
