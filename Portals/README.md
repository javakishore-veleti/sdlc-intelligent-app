# Portals

Angular front-end applications for the SDLC Intelligent App.

| Portal | Users | Purpose |
|---|---|---|
| **[sdlc-admin](./sdlc-admin)** | Administrators | Register business applications; define sprints and agendas |
| **[sdlc-nexus](./sdlc-nexus)** | Developers, Product Owners, Product Managers, Project Managers | Upload sprint PDFs, browse applications/sprints, and use the chatbot assistant |

`sdlc-nexus` is the central hub ("nexus") where all customer roles converge to work
with the team's sprint knowledge.

See [`../PRD.md`](../PRD.md) for personas and functional requirements.
