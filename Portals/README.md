# Portals

Angular front-end applications for the SDLC Intelligent App.

| Portal | Users | Purpose |
|---|---|---|
| **softwaredev-admin** | Administrators | Register business applications; define sprints and agendas |
| **requirements-engineering** | Developers, Product Owners, Product Managers, Project Managers | Upload sprint PDFs, browse applications/sprints, and use the chatbot assistant |

See [`../PRD.md`](../PRD.md) for personas and functional requirements.
